## 📖 Commands Covered

<img align="right" src="https://media.giphy.com/media/3oKIPnAiaMCJ8dO8dW/giphy.gif" width="120" alt="Cat Studying"/>

### 🔧 Setup & Config
```bash
git --version                          # Check Git version
git config --global user.name "Name"   # Set your name
git config --global user.email "Email" # Set your email
```

### 📁 Create & Track
```bash
git init                    # Initialize a new repo
git status                  # Check file status
git add .                   # Stage all changes
git commit -m "message"     # Commit with a message
```

### 🌐 Remote & Push
```bash
git remote add origin <url>   # Link to GitHub
git push -u origin main       # Push to remote
git clone <url>               # Clone a repo
git pull origin main          # Pull latest changes
```

<br clear="right"/>

### 🌿 Branching & Merge
```bash
git branch feature-name       # Create a branch
git checkout feature-name     # Switch to branch
git merge feature-name        # Merge branch into main
git branch -d feature-name    # Delete a branch
```

### ⏪ Undo Mistakes
```bash
git restore <file>            # Discard changes
git reset HEAD~1              # Undo last commit (keep files)
git revert <commit-hash>      # Create a reverse commit
```

### 🧰 Handy Tools
```bash
git stash                     # Stash uncommitted work
git stash pop                 # Restore stashed work
git diff                      # See what changed
git log --oneline             # View compact history
```

## 🌷 Daily Workflow Cheat Sheet

```bash
# ☀️ Morning routine — start fresh!
git pull origin main
git checkout -b feature/cute-update

# ✏️ Do your work, then...
git add .
git commit -m "✨ add adorable new feature"

# 🚀 Ship it!
git push origin feature/cute-update

# 🎉 After merge, clean up
git checkout main
git pull origin main
git branch -d feature/cute-update
```
