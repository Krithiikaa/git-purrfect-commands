## ✨ 𝐹𝑒𝒶𝓉𝓊𝓇𝑒𝓈
- 🎨 **Beautiful UI:** Elegant fonts, soft colors, and a dotted background.
- 📖 **Interactive Sidebar:** Easy navigation through all your favorite Git commands.
- 🐾 **Purr-fect Explanations:** Simple, beginner-friendly command descriptions.
- 📋 **One-Click Copy:** Easily copy code snippets to paste right into your terminal!

<br/>

## 🐱 𝐻𝑜𝓌 𝓉𝑜 𝒰𝓈𝑒
It's just one file! No build steps, no complicated setup.
1. Download or clone this repository.
2. Open `index.html` in your favorite browser.
3. Enjoy the cuteness while you learn Git! 💖
